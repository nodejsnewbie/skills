# 技能介绍

此技能旨在帮助教师自动化地依据《Java高级编程》双向细目表（命题要求）和复习大纲，抽取PPT中的教学内容来生成一套格式标准的期末考试试卷及配套参考答案。

# 前置依赖

1. **源数据读取**：
   - 教学PPT目录：`E:\repo\computer-course-materials\Java高级编程\《软件设计模式（Java版）》电子课件(PPT)`
   - 试卷纸模板：`E:\repo\computer-course-materials\Java高级编程\Java高级编程\试卷模板\1 试卷纸-A卷.doc`
   - 答题纸模板：`E:\repo\computer-course-materials\Java高级编程\Java高级编程\试卷模板\2 答题纸-A卷.doc`
   - 参考大纲：`E:\repo\computer-course-materials\Java高级编程\Java高级编程\试卷模板\Java高级编程复习大纲.docx`

# 执行流程

## 步骤 1：理解并解析双向细目表

根据《命题双向细目表》，我们需要出具总分为 100 分的试卷，分值和题型分布必须严格遵循如下规则：

- **选择题**：共29分。分布为：第一章2分、第二章4分、第三章2分、第四章4分、第五章2分、第六章3分、第七章10分、第八章2分。
- **填空题**：共12分。分布为：第一章2分、第二章4分、第五章2分、第六章2分、第七章2分。
- **判断题**：共6分。分布为：第一章1分、第二章3分、第四章1分、第五章1分。
- **简答题**：共30分。分布为：第一章5分、第二章5分、第三章5分、第四章5分、第六章5分、第八章5分。
- **综合题**：共23分。分布为：第三章10分、第五章3分、第八章10分。

## 步骤 2：提取并整理教学内容

1. 浏览和提取PPT课件内容及复习大纲，梳理每一章（按照双向细目表对应的章节，如“软件设计模式基础”、“创建型模式”、“结构型模式”、“行为型模式”等）的核心知识点集。
2. 对于 **选择题、填空题、判断题**，侧重于基本概念、特点、模式分类的认知与理解。
3. 对于 **简答题**，提取理论定义、优缺点分析及核心应用场景。
4. 对于 **综合题**，围绕UML类图结构、核心设计代码、各模式之间的结合应用和系统设计实例进行设计（如第三章、第五章、第八章）。

## 步骤 3：试卷题目生成与编排

针对前面提炼出的每章知识点结构，按照以下规则编拟具体题目：

1. 每道题目必须标注其对应的章节出处，以核实验证分布准确性。
2. 题目难度应区分“认知、理解、应用、分析、综合”层次。例如：简答题侧重“理解”，综合题需侧重“应用”及“综合”。
3. 题目设计不应偏离核心PPT及大纲知识面。

## 步骤 4：套用试卷模板，排版与格式化

将上一步生成的题目内容套用进模板结构中，分为：
一、选择题（每题X分，共29分）
二、填空题（每空X分，共12分）
三、判断题（每题X分，共6分）
四、简答题（每题5分，共30分）
五、综合题（共23分，根据具体题意设定每题分值）

最终生成两个文件直接输出：

1. **生成试卷.md**：排版精良的题干卷，包含答题纸的部分格式。
2. **参考答案及评分标准.md**：要求包含题目的详细解题步骤、标准答案和严谨的给分点。
